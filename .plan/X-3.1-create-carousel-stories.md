# Task 3.1: Create Carousel.stories.tsx

**Phase**: 3 - Storybook Stories + Vitest Interaction Tests

**Description**: Create Storybook stories for Carousel component with play function interaction tests

**Outcome**: `src/components/Carousel/Carousel.stories.tsx` with multiple stories and interaction tests

**Implementation**:

Create `src/components/Carousel/Carousel.stories.tsx`:

Mock data array of `ContentItem[]` defined once in a `__mocks__/movies.ts` fixture.

- `Default` story: renders carousel with mocked movies, `focusedMovie: 0`
- `NavigateRight` story with play function that tests the `onFocusChange` callback:
  ```typescript
  import { expect, fn, userEvent, waitFor, within } from '@storybook/test';

  export const NavigateRight: Story = {
    args: {
      movies: mockMovies,
      focusedMovie: 0,
      onFocusChange: fn(),
      onSelect: fn(),
    },
    play: async ({ args, canvasElement }) => {
      const canvas = within(canvasElement);
      // Initial: first item is focused
      await expect(canvas.getByTestId('carousel-item-0')).toHaveAttribute('data-focused', 'true');
      // Press right arrow
      await userEvent.keyboard('{ArrowRight}');
      // Assert callback was called with index 1
      await waitFor(() => {
        expect(args.onFocusChange).toHaveBeenCalledWith(1);
      });
    },
  };
  ```
- `FocusedAtThird`: renders with `focusedMovie: 2` to verify scroll position
- `NavigateLeftBoundary`: `focusedMovie: 0`, press Left, assert `onFocusChange` called with `-1` (store clamps, but component just calls the callback)
