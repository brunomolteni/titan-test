# Task 2.1: Implement CarouselItem component

**Phase**: 2 - Components

**Description**: Implement `CarouselItem.tsx` as a pure functional component. Props only, no store access.

**Outcome**: `src/components/Carousel/CarouselItem.tsx` implemented as pure component

**Implementation**:

Create `src/components/Carousel/CarouselItem.tsx`:
```typescript
interface CarouselItemProps {
  movie: ContentItem;
  isFocused: boolean;
  onSelect: (movie: ContentItem) => void;
  ref?: React.Ref<HTMLLIElement>;  // React 19: ref as prop
}
```

- Renders `<li>` with `<img src={movie.images.artwork_portrait}>`, `data-focused={isFocused}`
- Conditional BEM modifier: `isFocused ? styles['item--focused'] : ''`
- Calls `onSelect(movie)` on click/Enter
