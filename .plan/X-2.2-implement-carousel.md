# Task 2.2: Implement Carousel component

**Phase**: 2 - Components

**Description**: Implement `Carousel.tsx` as a pure functional component. Props only, no store access.

**Outcome**: `src/components/Carousel/Carousel.tsx` implemented with keyboard navigation and smooth scrolling

**Implementation**:

Create `src/components/Carousel/Carousel.tsx`:
```typescript
interface CarouselProps {
  movies: ContentItem[];
  focusedMovie: number;
  onFocusChange: (index: number) => void;
  onSelect: (movie: ContentItem) => void;
}
```

- Renders `<section>` > `<ul>` > maps `CarouselItem` children
- `useEffect` for `keydown` listener: ArrowLeft calls `onFocusChange(focusedMovie - 1)`, ArrowRight calls `onFocusChange(focusedMovie + 1)` (clamping is handled by the store)
- `useEffect` on `focusedMovie` change pins focused item to leftmost:
  ```typescript
  useEffect(() => {
    const el = itemRefs.current[focusedMovie];
    if (!el || !listRef.current) return;
    listRef.current.scrollTo({ left: el.offsetLeft, behavior: 'smooth' });
  }, [focusedMovie]);
  ```
- Container: `overflow-x: auto` with hidden scrollbar
