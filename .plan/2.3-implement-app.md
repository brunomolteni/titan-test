# Task 2.3: Implement App component

**Phase**: 2 - Components

**Description**: Implement `App.tsx` -- the only component that reads the store

**Outcome**: `src/App.tsx` implemented, connecting store to pure Carousel component

**Implementation**:

Create `src/App.tsx`:
```typescript
import { useMoviesStore } from './stores/useMoviesStore';
import { useShallow } from 'zustand/shallow';

export default function App() {
  const { movies, focusedMovie, fetchMovies, focusMovie, selectMovie } =
    useMoviesStore(useShallow((s) => ({
      movies: s.movies,
      focusedMovie: s.focusedMovie,
      fetchMovies: s.fetchMovies,
      focusMovie: s.focusMovie,
      selectMovie: s.selectMovie,
    })));

  useEffect(() => { fetchMovies(); }, [fetchMovies]);

  return (
    <main className={styles.app}>
      <h1 className={styles.app__title}>TitanTest</h1>
      <Carousel
        movies={movies}
        focusedMovie={focusedMovie}
        onFocusChange={focusMovie}
        onSelect={selectMovie}
      />
    </main>
  );
}
```

- `preconnect('https://eu.acc01.titanos.tv')` in `main.tsx`
- Store is accessed ONLY here; Carousel and CarouselItem are fully props-driven
