# Task 0.7: Set up CSS foundation

**Phase**: 0 - Project Scaffold

**Description**: Create CSS foundation with reset, design tokens (colors, spacing, typography), and global styles

**Outcome**: Complete CSS foundation with dark Netflix-inspired theme

**Implementation**:

Create `src/styles/reset.css` -- modern CSS reset

Create `src/styles/tokens/colors.css`:
```css
:root {
  --color-bg: #141414;
  --color-surface: #1a1a2e;
  --color-focus-ring: #e50914;
  --color-text: #ffffff;
  --color-text-muted: #b3b3b3;
  --radius-md: 4px;
}
```

Create `src/styles/tokens/spacing.css`: `--spacing-xs: 0.25rem` through `--spacing-2xl: 3rem`

Create `src/styles/tokens/typography.css`: system font stack, `--font-size-*` scale

Create `src/styles/globals.css`: imports all above, sets `body { background: var(--color-bg); color: var(--color-text); }`

Import `globals.css` in `src/main.tsx` AND `.storybook/preview.ts`
