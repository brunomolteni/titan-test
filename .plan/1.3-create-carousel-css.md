# Task 1.3: Create Carousel CSS Modules

**Phase**: 1 - Types + Store + CSS (Branch B)

**Description**: Create CSS Modules for Carousel and CarouselItem components using BEM methodology and design tokens

**Outcome**: `Carousel.module.css` and `CarouselItem.module.css` created

**Implementation**:

Create `src/components/Carousel/Carousel.module.css`:
```css
.carousel {
  position: relative;
  width: 100vw;
  overflow: hidden;
}
.carousel__list {
  display: flex;
  gap: var(--spacing-md);
  padding: var(--spacing-lg) 0;
  overflow-x: auto;
  scrollbar-width: none;
}
.carousel__list::-webkit-scrollbar { display: none; }
```

Create `src/components/Carousel/CarouselItem.module.css`:
```css
.item {
  flex: 0 0 auto;
  outline: none;
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}
.item--focused {
  transform: scale(1.08);
  box-shadow: 0 0 0 3px var(--color-focus-ring);
  z-index: 1;
}
.item__image {
  width: 160px;
  aspect-ratio: 2 / 3;
  object-fit: cover;
  border-radius: var(--radius-md);
  display: block;
}
```
