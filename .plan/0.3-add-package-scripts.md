# Task 0.3: Add package.json scripts

**Phase**: 0 - Project Scaffold

**Description**: Add npm/bun scripts for development, build, testing, and Storybook

**Outcome**: `package.json` contains all necessary scripts

**Implementation**:

Add to `package.json`:
```json
{
  "scripts": {
    "dev": "bun ./index.html",
    "build": "bun build ./index.html --outdir=dist --minify",
    "storybook": "storybook dev -p 6006",
    "build-storybook": "storybook build",
    "test": "vitest --project storybook",
    "test:e2e": "playwright test",
    "test:visual": "playwright test --project visual"
  }
}
```
